REPORT z.
TABLES: SFLIGHT.

TYPES: BEGIN OF SFLIGHTTYPE,
          CARRID TYPE  SFLIGHT-CARRID ,
          CONNID TYPE SFLIGHT-CARRID ,
       END OF SFLIGHTTYPE.
 DATA: Itab_SFLIGHT    TYPE  STANDARD TABLE OF SFLIGHTTYPE INITIAL SIZE 0.
 DATA: WA TYPE  SFLIGHTTYPE.      .

SELECTION-SCREEN BEGIN OF BLOCK BLK1 WITH FRAME TITLE TEXT-100.
  PARAMETERS: P_CARRID TYPE SFLIGHT-CARRID.
SELECTION-SCREEN END OF BLOCK BLK1.

AT SELECTION-SCREEN ON P_CARRID.
IF P_CARRID IS INITIAL.
  MESSAGE E001(Z104329MSGCLASS) WITH 'CARRID SHOULD NOT BE BLANK'.
ENDIF.
IF P_CARRID NE SPACE.
  SELECT CARRID  CONNID  FROM SFLIGHT
    INTO TABLE Itab_SFLIGHT WHERE CARRID EQ P_CARRID.

IF SY-SUBRC NE 0.
    MESSAGE E001(Z104329MSGCLASS) WITH 'NO RECORD PRESENT'.
ENDIF.
ENDIF.

START-OF-SELECTION.
LOOP AT Itab_SFLIGHT INTO WA.
  WRITE: /  WA-CARRID,WA-CONNID.
ENDLOOP.
