  PARAMETERS: NUMBER1 TYPE I,
              NUMBER2 TYPE I,
              NUMBER3 TYPE I.

SELECTION-SCREEN BEGIN OF BLOCK PART2 WITH FRAME.
  PARAMETERS: NUMBER4 TYPE I,
              NUMBER5 TYPE I,
              NUMBER6 TYPE I.

  IF NUMBER3 LT NUMBER2 OR
     NUMBER3 LT NUMBER1 OR
     NUMBER2 LT NUMBER1.
     MESSAGE E001(Z204329MSGCLASS) WITH 'Enter numbers in ascending order'.
  ENDIF.

  IF NUMBER6 LT NUMBER5 OR
     NUMBER6 LT NUMBER4 OR
     NUMBER5 LT NUMBER4.

  ENDIF.
