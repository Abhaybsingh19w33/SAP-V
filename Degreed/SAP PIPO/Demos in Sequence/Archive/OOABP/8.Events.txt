REPORT ZAT_OOAP_DEM008.


CLASS C1 DEFINITION.
  PUBLIC SECTION.
  EVENTS: E1.
  METHODS: M1 FOR EVENT E1 OF C1.
  METHODS: T1.
  ENDCLASS.

  CLASS C1 IMPLEMENTATION.
    METHOD M1.
      WRITE 'EVENT handler method'.
      ENDMETHOD.
      METHOD T1.
        WRITE 'Raise the event'.
        RAISE EVENT E1.

        ENDMETHOD.
    ENDCLASS.


    START-OF-SELECTION.
    DATA OBJ TYPE REF TO C1.
    CREATE OBJECT OBJ.
    SET HANDLER OBJ->M1 FOR OBJ.
    CALL METHOD OBJ->T1.
