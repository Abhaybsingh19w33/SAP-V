Step1 
To create a Type pool
Start a new session
type /ose11
select type group enter the name as 
ZVKTP
Enter short text
save

Go to SE38 to implement the type pool created above.
TYPE-POOL ZVKTP.
*DATA:ZVKTP_DOLLARS(16)	TYPE P DECIMALS 2,
*      ZVKTP_LIRA(16)    TYPE P DECIMALS 2.
*Cannot have data here

TYPES:ZVKTP_DOLLARS(16)	TYPE P DECIMALS 2,
      ZVKTP_LIRA(16)    TYPE P DECIMALS 2.
CONSTANTS:
 	ZVKTP_WARNING_THRESHOLD	TYPE I VALUE 5000,
  ZVKTP_AMALGAMATION_DATE	TYPE  SY-DATUM VALUE '20030827'.TION_DATE	TYPE  SY-DATUM VALUE '20030827'.

--------------------------------------------------------------
REPORT Z.
TYPE-POOLS ZVKTP.
DATA:BEGIN OF AMERICAN_SUMS,
        PETTY_CASH TYPE ZVKTP_DOLLARS,
        PAY_OUTS   TYPE ZVKTP_DOLLARS,
        LUMP_SUMS  TYPE ZVKTP_DOLLARS,
     END OF AMERICAN_SUMS.


DATA:  BEGIN OF ITALIAN_SUMS,
        PETTY_CASH TYPE ZVKTP_LIRA,
        PAY_OUTS   TYPE ZVKTP_LIRA,
        LUMP_SUMS  TYPE ZVKTP_LIRA,
      END OF ITALIAN_SUMS.

AMERICAN_SUMS-PAY_OUTS  = '9500.03'.
ITALIAN_SUMS-LUMP_SUMS  = 9555.

IF  AMERICAN_SUMS-PAY_OUTS > ZVKTP_WARNING_THRESHOLD.
  WRITE : / 'WARNING', AMERICAN_SUMS-PAY_OUTS,
   'EXCEED THRESHOLD ', ZVKTP_WARNING_THRESHOLD.
ENDIF.
